Situational Awareness Applicaton using JavaScript and Leaflet

This is my final project for GEOG 575. JavaScript and Leaflet were used to create this map that combines parcel data and business points for Cottage Grove, WI. The data was converted to geoJSON format in a WGS 1984 Geographic Coordinate System. This web application includes a Mapzen geocoder and an optional Weather Radar layer from Iowa State University.
